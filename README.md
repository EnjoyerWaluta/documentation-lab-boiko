# College Schedule Website

## Опис
College Schedule Website — це вебсайт, який дозволяє студентам і викладачам швидко переглядати розклад занять у коледжі.

Система допомагає уникнути плутанини з розкладом та надає доступ до актуальної інформації про заняття.

## Основні функції
- перегляд розкладу занять
- пошук розкладу за групою
- перегляд розкладу викладача
- перегляд аудиторій
- оновлення розкладу адміністрацією

## Для кого цей продукт
Цей вебсайт призначений для:
- студентів коледжу
- викладачів
- адміністрації навчального закладу

## Приклад використання
Студент може обрати свою групу і переглянути розклад занять на день або на тиждень.
