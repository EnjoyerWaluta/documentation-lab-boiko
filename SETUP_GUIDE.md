# Setup Guide

## Необхідні інструменти
Для використання системи потрібні:

- комп'ютер або смартфон
- браузер (Chrome, Firefox, Edge)
- підключення до інтернету

## Кроки запуску системи

1. Відкрити браузер.
2. Перейти на вебсайт розкладу коледжу.
3. На головній сторінці вибрати групу.
4. Натиснути кнопку "Переглянути розклад".

## Використання для адміністрації (умовно)

Адміністратор може:
- додавати нові заняття
- редагувати розклад
- видаляти заняття

## Перевірка роботи

Система працює правильно, якщо:
- сторінка відкривається без помилок
- список груп відображається
- після вибору групи показується розклад занять
